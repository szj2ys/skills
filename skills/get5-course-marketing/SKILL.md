# 得到课程运营与推广

基于《得到品控手册》的课程运营方法论，将课程上新从"信息发布"升级为"价值传递"，让用户成为课程的传播者。

## 核心能力一：课程上新运营五要素

### 一、老师人设立得住

在上新运营的预热期，老师人设是关键要素。"立得住"需要满足三个条件：

| 条件 | 要求 |
|------|------|
| **必须是正面的** | 能够为课程加分，对课程销量有帮助 |
| **放大显著优点** | 不能硬拗造型，要基于老师真实特点 |
| **可被用户感知** | 不会产生误解和歧义 |

**知识向人设类型**：

| 人设类型 | 含义 | 案例 |
|----------|------|------|
| **同行者** | 与用户并肩前行的伙伴 | 蔡钰"商业参考"的课代表人设 |
| **学霸同学** | 学习能力强的优秀同学 | 万维钢"精英日课"的人设 |
| **邻家叔叔** | 亲切可信赖的长辈 | 和菜头"成年人修炼手册"的人设 |
| **军师** | 提供策略建议的智囊 | 熊太行"关系攻略"的人设 |
| **见多识广的长者** | 阅历丰富的导师 | 吴军"硅谷来信"的形象 |

**实践要点**：在内容生产和上新策划时，与老师充分沟通，预设"知识向人设"，确保人设真实、正面、可感知。

---

### 二、购买理由够具体

找到课程对用户的具体利益点，从用户角度回应他们对新课关心的问题。不是简单把所有信息摊给用户，而是要"翻译"这些信息对用户意味着什么。

**关键翻译维度**：

| 维度 | 翻译方向 | 示例 |
|------|----------|------|
| **价格** | 对用户来说是否超值 | "每天不到1元钱" |
| **讲数及每讲时长** | 是否能帮用户省时间 | "每天10分钟，轻松掌握" |
| **学完效果** | 能够达到什么状态 | "站在高处，重新理解财富" |

**经典案例**：

- 《香帅的北大金融学课》："站在高处，重新理解财富"
- 《傅骏·美食鉴赏15讲》："提升你享受幸福的能力"

---

### 三、详情页可感知

课程详情页由三个板块组成："主理人介绍"→"课程亮点"→"课程表"。

**设计要点**：

| 板块 | 核心要求 |
|------|----------|
| **课程亮点** | 必须包含"学完会获得什么"和"谁适合学习这门课" |
| **课程表介绍** | 给用户做"翻译"：为什么要讲这些内容，按什么逻辑设计，建立学习地图 |
| **购买障碍清除** | 不只是展示信息，还要告知用户这些信息对他们的意义 |

---

### 四、分享海报易传播

分享海报是把购买理由变成图文结合的呈现方式，是课程向外部营销的重要手段。

**核心原则**：

- 好的分享海报能让信息高效传递
- 激发用户通过朋友圈、微信群、社交媒体等渠道传播
- **关键洞察**：分享海报不是课程研发者要跟用户说什么，而是一位用户想利用这张海报跟他的朋友说什么

---

### 五、发刊词姿势对

发刊词是课程上线后用户第一时间接触到的正式内容，是"第0讲"。

**正确定位**：

- 不是书的"自序"，不是作者表达个人感受和感谢
- 而是**销售文案**，任务是向全站用户发出邀请或作出承诺
- 要做好两点：充分介绍老师，让外面的人进来

**案例**：《段涛·产科医学课》的发刊词针对不同群体喊话：

| 目标群体 | 喊话方向 |
|----------|----------|
| 自己打算生育的 | 提供专业知识，消除焦虑 |
| 已经生育完毕的 | 回顾总结，提升认知 |
| 身边有人正在准备生育的 | 成为身边人的知识顾问 |

## 核心能力二：AI辅助制作宣传物料

### AI制作课程宣传视频

**四个关键要素**：

| 要素 | 要求 |
|------|------|
| **评估合适的课程** | 不是所有课程都适合用AI制作视频 |
| **明确AI工具定位** | 核心不是追求"技术感"，而是找到高性价比解决方案 |
| **找准参照物** | 提供风格参考和明确要求 |
| **留出人工精修空间** | AI负责发散，人负责收敛 |

**推荐工具组合**：

| 环节 | 工具 |
|------|------|
| **文案策划** | ChatGPT、DeepSeek |
| **视觉画面生成** | 即梦、Midjourney、可灵 |
| **音乐制作** | Suno |
| **剪辑包装** | 剪映 |

### AI制作推广海报

**形式包括**：

- **场景海报**：以图像为主要冲击力的海报
- **文字型海报**：以课程金句为主的海报

**实际案例**：《施展·中国史纲》空间篇课程宣传九宫格海报：

- 视觉重心是金句
- 背景图用AI生成
- 元素围绕山脉、河流、关隘等地理元素或历史意象展开
- 避免出现近景人物喧宾夺主

## 核心能力三：AI辅助处理用户留言

### 核心原则：人机协同而非AI替代

利用AI辅助处理用户留言，核心在于实现"人机协同"而非"AI替代"。

### 实际应用机制

**AI筛选优先级**：

1. **推荐精选**：可互动的留言
2. **提问咨询**：需要回复的问题
3. **反馈评价**：对课程的反馈和评价

**处理原则**：

- 提醒课程主理人/主编优先处理这些留言
- 不是对所有留言都回复，更不是用固定模板机械回复

### 双向进化机制

| 方向 | 机制 |
|------|------|
| **人→AI** | 课程主理人/编辑通过对AI筛选结果的持续监督与调整，提升系统适应性 |
| **AI→人** | AI通过学习反馈模式，逐渐形成更贴合课程风格的处理方式 |

**最终目标**：确保技术工具始终服务于课程内容的核心价值。

---

## 使用方式

根据用户的具体需求，选择对应模块进行运营和推广：

- **课程上新运营**：使用"五要素法"逐项检查：老师人设、购买理由、详情页、分享海报、发刊词
- **购买理由提炼**：使用"翻译维度"（价格、时间、效果）将课程信息转化为用户利益点
- **详情页设计**：确保包含"主理人介绍"→"课程亮点"→"课程表"三个板块，并做"翻译"
- **分享海报策划**：以"用户想跟朋友说什么"为核心洞察设计海报
- **发刊词撰写**：定位为销售文案，向全站用户发出邀请或作出承诺
- **AI宣传物料制作**：使用推荐工具组合，遵循"AI发散+人工收敛"原则
- **用户留言处理**：AI筛选+人工回复，建立双向进化机制

输出格式建议：Markdown 表格或清单，便于逐项对照使用。
