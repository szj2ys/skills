# szj2ys/skills

A categorized collection of skills for AI agents.

## Installation

```bash
npx skills add szj2ys/skills
```

## Skill Categories

- **`design/`**: UI/UX, design systems, visual taste, image generation, animation (30 skills)
- **`engineering/`**: Coding, testing, reviewing, deployment, React, Next.js, CI/CD (74 skills)
- **`marketing/`**: SEO, GEO, ads, copywriting, analytics, growth, sales enablement (65 skills)
- **`toolchains/`**: Language/framework-specific skills — JS/TS, Python, Rust, Go, Elixir, PHP, Java, VB, UI, platforms, databases (123 skills)
- **`universal/`**: Cross-stack skills — debugging, security, testing, verification, infrastructure, orchestration (41 skills)
- **`devtools/`**: Cursor plugins, IDE workflows, pstack principles, team collaboration tools (65 skills)
- **`gstack/`**: iOS development, design workflow, deployment, QA, browser automation, brainstorming (59 skills)
- **`product-management/`**: PM fundamentals, growth, startup, engineering leadership (7 skills)
- **`productivity/`**: Prompt enhancement, writing, brainstorming, agent skills management (10 skills)
- **`learning-and-growth/`**: Self-improvement, writing plans, interviewing (4 skills)
- **`in-progress/`**: Skills under active development (5 skills)
- **`personal/`**: Personal or highly customized skills (2 skills)
- **`misc/`**: Experimental or narrowly focused skills (11 skills)
- **`research/`**: Research and news monitoring (1 skill)
- **`documents/`**: Document generation — PDF, HTML slides (3 skills)

### Standalone

- **`agent-browser/`**: Browser automation CLI for AI agents (1 skill)
